Welcome to Adobe Substance 3D Sampler Scripting documentation
=============================================================

The `Substance 3D Sampler` Python API allows users to write their plugins and scripts to automate their workflow and customize their interface to integrate efficiently Sampler in their pipeline.

A `Substance 3D Sampler` script is a standard Python module imported in application through Preferences window.

A `Substance 3D Sampler` plugin contains both scripting and UI elements, described in QML, and packed into a zip archive.
