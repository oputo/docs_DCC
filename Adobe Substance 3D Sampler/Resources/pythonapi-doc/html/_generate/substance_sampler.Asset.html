  • w_resolution (int) – Width resolution in pixel

  • h_resolution (int) – Height resolution in pixel

  • path (str) – Export folder path

  • name (str) – Name of the exported file(s): By default: name is the Asset’s name

  • format (ExportFormat) – Format Enum (sbsar, sbs, png, …). By default: sbsar

  • preset (Resource) – Resource representing export preset. By default: empty preset

  • channels (List[str]) – list of channels. By default: all channels of current workflow

  • subfolder (bool) – Export in a subfolder. False by default

  • physical_ratio (bool) – Export with physical ratio. False by default

  • overwrite (bool) – Overwrite if files already exist. True by default

Export 3D object asset : both mesh and material

Parameters:

  • path (str) – Export folder path

  • name (str) – Name of the exported file(s): By default: name is the Asset’s name

  • subfolder (bool) – Export in a subfolder. False by default

  • mesh_format (MeshExportFormat) – Mesh Format Enum (usd, fbx, obj, …). By default: usd

  • material_format (MaterialExportFormat) – Material Format Enum (sbsar, sbs, png, …). By default: sbsar

  • material_type (MaterialType) – The type of material (standard, decal, atlas). By default: standard

  • preset (Resource) – Resource representing export preset. By default: empty preset

  • w_resolution (int) – Overriden width resolution in pixel for material generation. If not set, use layer output resolution

  • h_resolution (int) – Overriden height resolution in pixel for material generation. If not set, use layer output resolution

  • channels (List[str]) – list of channels. By default: all channels of current workflow

  • physical_ratio (bool) – Export with physical ratio. False by default. Only used if both w_resolution and h_resolution are set

  • overwrite (bool) – Overwrite if files already exist. True by default

  • controller (ExportController) – Optional export controller. None by default

Returns:
