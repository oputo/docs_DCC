<static> isBusy()

Check if Substance 3D Painter is currently busy.
If busy, the project cannot be saved at the moment. The application may be busy because no project is in edition state, or a long process such as baking/export/unwrap process is ongoing. The corresponding busyStatusChanged event is fired when the busy state changes.

<static> save( [file] [, mode])

Save the current project.
Save is disabled when Substance 3D Painter is busy and will throw a ProjectError. See 'isBusy'.
Parameters:
Name Type Argument Default Description
file string <optional>
 Save url. Leave empty to save in place.
mode alg.project.SaveMode <optional>
 Incremental Save mode.
Throws:
alg.project.ProjectError
Examples
// Save in place, in Incremental mode
     alg.project.save()
     // We have the same behavior than calling save with the default values
     alg.project.save("", alg.project.SaveMode.Incremental)
     
// Save as
     alg.project.save("file:///c:/Documents/project.spp")
     
// Saving multiple times at the same place a project in Incremental mode can create fragmentation inside the project file.
     // This increase the file size on the disk.
     // Performing a Full save will remove this fragmentation issue.
     // To do that, we have to save in place (empty path), but specifying the Full mode.
     alg.project.save("", alg.project.SaveMode.Full);

<static> saveAsCopy(file [, mode])

Save a copy of the current project while keeping the original version opened.
This can be used to save backups of the opened project without modifying the original file.
Save is disabled when Substance 3D Painter is busy and will throw a ProjectError. See 'isBusy'.
Parameters:
Name Type Argument Default Description
file string Save url.
mode alg.project.SaveMode <optional>
 Incremental Save mode.
Throws:
alg.project.ProjectError
Example
// Save
     alg.project.save("file:///c:/Documents/project.spp")
     // Save as copy
     alg.project.saveAsCopy("file:///c:/Documents/project_copy.spp")
     // alg.project.url() will output "file:///c:/Documents/project.spp"
     alg.log.info(alg.project.url());
