<static> setColorLutResource(newUrl)

Set the color profile LUT resource URL of the active project
Parameters:
Name Type Description
newUrl string The new color profile LUT URL
Throws:
alg.display.DisplaySettingsError
Example
// Log the resource url of the color LUT currently used in the opened project
alg.log.info("Current color LUT: " + alg.display.getColorLutResource())
// Look for the sepia color LUT
var res = alg.resources.findResources("allego*", "sepia")[0]
// Set sepia as the new color LUT.
alg.display.setColorLutResource(res)
// Log the resource url of the newly used color LUT
alg.log.info("New color LUT: " + alg.display.getColorLutResource())

// Output example
// Current color LUT: resource://starter_assets/film_noir?version=92858924bb9cc52b0d8ae876b045a91eeaef9055.image
// New color LUT: resource://starter_assets/sepia?version=aa496c84d1c09d4f17f1778bff7215f9a1943f2c.image
