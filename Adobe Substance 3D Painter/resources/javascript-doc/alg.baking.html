<static> setTextureSetBakingParameters(textureSetName, parameters)

Set baking parameters of a texture set of the current project
Parameters:
Name Type Description
textureSetName string The texture set name
parameters object The baking parameters to merge as a JSON object
Throws:
alg.baking.BakingError
Example
// get the texture set baking parameters
     var parameters = alg.baking.textureSetBakingParameters("textureSetName")
     // disable ambient occlusion
     parameters.definitions.Ambient_occlusion.enabled = false
     // it is also possible to do it this way
     parameters["definitions"]["Ambient_occlusion"]["enabled"] = false
     // set the new texture set baking parameters
     alg.baking.setTextureSetBakingParameters("textureSetName", parameters)
