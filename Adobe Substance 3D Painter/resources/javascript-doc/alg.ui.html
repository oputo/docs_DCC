<static> addAction(menu, text, tooltip, idleImage, hoverImage)

Create an action using a text and tooltip. Associate with an icon created with the provided images. Register it in the given Painter menu. The library keeps the ownership of the returned action ie caller doesn't have to manage object destruction.
Parameters:
Name Type Description
menu alg.ui.AppMenu Menu in Painter where to register the action.
text string Displayed text of the action.
tooltip string Tooltip of the action.
idleImage url Idle image of the action icon.
hoverImage url Hover image of the action icon.
Throws:
If the images cannot be loaded.
Type
alg.ui.UiError
Returns:
The created action. Library keeps ownership.
Type
QAction
