<static> save(dataPath, filePath [, mapInfo])

Render and save a part of the document to a file.
Parameters:
Name Type Argument Description
dataPath Array.<object>

Description of the object to be rendered.

Generic form of the data path: ["Texture set name", "Optional stack name", "Optional group name", ..., "Optional layer name", "Map name"].

It is also possible to use a layer uid to generate the data path: [Layer uid (number), "Map name"].

Valid map names are: mask, basecolor, base color, height, specular, opacity, emissive, displacement, glossiness, roughness, anisotropylevel, anisotropy level, anisotropyangle, anisotropy angle, transmissive, reflection, ior, metallic, normal, ambientocclusion, user0, user 0, user1, user 1, user2, user 2, user3, user 3, user4, user 4, user5, user 5, user6, user 6, user7, user 7, [custom user names].
filePath string The file where the image will be saved. Image format is based on file extension.
mapInfo alg.mapexport.MapInfo <optional>
 Control export parameters
Throws:
alg.mapexport.ExportError
Examples
// export the "base color" map from the "M1" texture set in jpg format
     alg.mapexport.save(["M1", "base color"], "c:/file_path.jpg")

     
// export the mask of a layer in png format
     alg.mapexport.save(["M1", "Group 1", "Layer 1", "mask"], "c:/file_path.png")

     
// export the mask of a layer specifying the uid
     alg.mapexport.save([24, "mask"], "c:/file_path.png")

     
// export without padding and a custom resolution
     alg.mapexport.save(
        ["M1", "base color"],
        "c:/file_path.jpg",
        {padding:"Transparent", dilation:0, resolution:[256,512]})

     
// explicitly disable dithering on normal/height maps when exporting with 8-bits file formats
     alg.mapexport.save(["M1", "normal"], "c:/file_path.tga", {dithering:"disabled"})

MapInfo

Type:
  • Object
Properties:
Name Type Argument Default Description
padding string <optional>
 "Infinite" Control the type of padding. Possible values are: "Passthrough", "Infinite", "Transparent", "Color", "Diffusion". Refer to the user documentation for meanings.
dilation int <optional>
 16 Padding dilation size in pixels (no effect if padding is "Passthrough" or "Infinite")
dithering string <optional>
 "enabled" When exporting a 16-bit texture as 8-bit, unwanted artefacts may arise. Enabling dithering can reduce them. Dithering is only applied to Height and Normal maps that are exported at 8-bit. Possible values are: "disabled", "enabled".
resolution Array.<int> <optional>
 Resolution of the rendered map (array of two power of two numbers), refer to application limitation for resolutions minimum and maximum. If not specified, uses the texture set resolution.
bitDepth int <optional>
 8 Bit per channel to use (could be one of 8, 16 or 32 bpc)
keepAlpha boolean <optional>
 false Keep the alpha channel of the exported map if true

StackPath
