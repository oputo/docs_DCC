<static> addChannel(textureSetName, type, format [, label])

Add a new channel to a texture set.
Parameters:
Name Type Argument Description
textureSetName string Name of the target texture set.
type string The channel identifier. See channelIdentifiers for details.
format string See channel format label property.
label string <optional>
 Label of the channel in case of User channel as type.
Throws:
alg.texturesets.TextureSetsError
Example
// add a scattering channel inside the 'DefaultMaterial' texture set
alg.texturesets.addChannel("DefaultMaterial", "scattering", "L8")

<static> editChannel(stackPath, type, format [, label])

Same as addChannel but edit the channel instead. See addChannel for parameters details.
Parameters:
Name Type Argument Description
stackPath alg.mapexport.StackPath Path to the stack.
type string The channel identifier. See channelIdentifiers for details.
format string See channel format label property.
label string <optional>
 Label of the channel in case of User channel as type.
Throws:
alg.texturesets.TextureSetsError
Examples
// edit a scattering channel inside the 'DefaultMaterial' texture set
alg.texturesets.editChannel("DefaultMaterial", "scattering", "L8")
     
// edit a channel in case of material layering
alg.texturesets.editChannel(["DefaultMaterial", "Mask"], "blendingmask", "L8")

// set the 'DefaultMaterial' texture set the current paintable texture set

<static> setResolution(textureSetNames, resolution)

Set the resolution of the specified texture sets.
Parameters:
Name Type Description
textureSetNames string | Array.<string> Name of the target texture sets.
resolution Array.<int> The wanted log2 resolution for the specified texture sets.
Throws:
alg.texturesets.TextureSetsError
Example
// change resolution of the 'DefaultMaterial' texture set to 4096x4096
alg.texturesets.setResolution("DefaultMaterial", [12,12])
// change resolution of the 'DefaultMaterial' and 'DefaultMaterial2' texture sets to 4096x4096
alg.texturesets.setResolution(["DefaultMaterial", "DefaultMaterial2"], [12,12])
