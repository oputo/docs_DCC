Substance 3D Painter plugin system

Extend Substance 3D Painter functionalities in javascript and qml.

Plugin installation

Create a folder in "Documents/Adobe/Substance 3D Painter/plugins" that contains at least a "main.qml" file. The folder name will be used as the plugin name. Plugins can be enabled/disabled from the Substance 3D Painter Plugins menu entry.

API Changes

1.1.17 (Substance 3D Painter 8.3.0)

  • Common parameters for baking is now instanciated per material as materialParameters.
  • commonBakingParameters now always throw an error. Use textureSetBakingParameters instead.
  • setCommonBakingParameters now always throw an error. Use setTextureSetBakingParameters instead.
  • textureSetBakingParameters now outputs common parameters for the texture set as materialParameters.
  • setTextureSetBakingParameters now allows to set common parameters for the texture set as materialParameters.
  • curvatureMethod now requires a texture set name.
  • setCurvatureMethod now requires a texture set name.
