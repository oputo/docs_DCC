<static> findResources(shelfName, resourceName)

Find resources in a shelf by name.
Parameters:
Name Type Description
shelfName string | RegExp Name of the shelf in which you want to search. When using a string, wildcard matching is used (see examples). Case Sensitive.
resourceName string | RegExp Name of the resource you want to find. When using a string, wildcard matching is used (see examples). Case Sensitive.
Throws:
If shelfName or resourceName aren't valid, or if shelfName doesn't match any shelves.
Type
alg.resources.ResourcesError
Returns:
The list of found resource urls alphabetically ordered.
Type
Array.<string>
Examples
// For this example consider we only have two existing resources named 'leather_bag/leather_bag' and
// 'leather_soft_grain/leather_soft_grain' in the starter_assets shelf.

// Perfect matching
var resourceName = "leather_bag/leather_bag"
var resources = alg.resources.findResources("starter_assets", resourceName)
alg.log.info(("Nb resources found for '%1': %2").arg(resourceName).arg(resources.length))
alg.log.info(JSON.stringify(resources[0], null, '  '))
// Will output:
// Nb resources found for 'leather_bag/leather_bag': 1
// "resource://starter_assets/leather_bag/leather_bag?version=00fccae0626fb9b90984d27c192a0ff70b3313be.sbsar"

// Perfect matching
resourceName = "leather_bag"
resources = alg.resources.findResources("starter_assets", resourceName)
alg.log.info(("Nb resources found for '%1': %2").arg(resourceName).arg(resources.length))
// Will output:
// Nb resources found for 'leather_bag': 0

// Wildcard matching
resourceName = "*leather_bag*"
resources = alg.resources.findResources("starter_assets", resourceName)
alg.log.info(("Nb resources found for '%1': %2").arg(resourceName).arg(resources.length))
alg.log.info(JSON.stringify(resources[0], null, '  '))
// Will output:
// Nb resources found for '*leather_bag*': 1
// "resource://starter_assets/leather_bag/leather_bag?version=00fccae0626fb9b90984d27c192a0ff70b3313be.sbsar"

// Regular expression matching
resourceName = /.*leather_bag.*$/
resources = alg.resources.findResources("starter_assets", resourceName)
alg.log.info(("Nb resources found for '%1': %2").arg(resourceName).arg(resources.length))
alg.log.info(JSON.stringify(resources[0], null, '  '))
// Will output:
// Nb resources found for '/.*leather_bag.*$/': 1
// "resource://starter_assets/leather_bag/leather_bag?version=00fccae0626fb9b90984d27c192a0ff70b3313be.sbsar"

     
// Get all the resources available in all the shelves.
var resources = alg.resources.findResources("*", "*")
alg.log.info("Nb resources found for shelf name '*' and resource name '*': "+resources.length)

// Get all the resources available in the 'starter_assets' shelf
resources = alg.resources.findResources("starter_assets", "*")
alg.log.info("Nb resources found for shelf name 'starter_assets' and resource name '*': "+resources.length)

// Get all the resources available in the shelves whose names start by 'allego'
resources = alg.resources.findResources("allego*", "*")
alg.log.info("Nb resources found for shelf name 'allego*' and resource name '*': "+resources.length)

// Get all the resources available in the shelves whose names contain 'allego'
resources = alg.resources.findResources("*allego*", "*")
alg.log.info("Nb resources found for shelf name '*allego*' and resource name '*': "+resources.length)

// Get all the resources available in the shelves 'starter_assets' and 'shelf'
var resources = alg.resources.findResources(/^(shelf|starter_assets)$/, "*")
alg.log.info("Nb resources found for shelf name '/^(shelf|starter_assets)$/' and resource name '*': "+resources.length)

}

"ERROR: Import failed because the usages ['shader'] aren't compatible with this resource."

<static> updateDocumentResources(oldUrl, newUrl)

Update resources of the currently opened project. Only Substances, Images and Particles can be updated. Careful this method will clear all your undo stacks.
Parameters:
Name Type Description
oldUrl string Resource url you want to replace in the document.
newUrl string The url to put in place of the urls matching oldUrl.
Throws:
If no project is opened, if oldUrl/newUrl aren't valid or if newUrl can't replace the old one.
Type
alg.resources.ResourcesError
Returns:
The urls that have been replaced in the document alphabetically ordered.
Type
Array.<string>
Example
// List all the resources used in the project
var resources = alg.resources.documentResources()
alg.log.info("Resources used inside the project: "+resources.length)
for(var i = 0; i < resources.length; ++i) {
    alg.log.info(resources[i])
}

// For each resource check if a different version exists in the same shelf.
// If so, update it.
alg.log.info("Try to find new version for these resources")
for(var i = 0; i < resources.length; ++i) {
    var resource = resources[i]
	var resourceInfo = alg.resources.getResourceInfo(resource)
    var matches = alg.resources.findResources(resourceInfo.shelfName, resourceInfo.name)
    for(var j = 0; j < matches.length; ++j) {
        var match = matches[j]
        var matchInfo = alg.resources.getResourceInfo(match)
        if(matchInfo.version == resourceInfo.version) continue
        alg.log.info("Updating %1 from version %2 to %3"
                     .arg(resourceInfo.name).arg(resourceInfo.version).arg(matchInfo.version))
        alg.resources.updateDocumentResources(resource, match)
    }
}

Type Definitions

ResourceInfo

Information on a resource. Some properties are only available if the resource currently exists (guiName, type, usages and shelfPath).
Type:
  • Object
Properties:
Name Type Argument Description
url string Url of the resource identifying it.
name string Name of the resource.
guiName string <optional>
 Resource name displayed in the gui.
version string Version of the resource. Unique value computed using the resource data.
type string <optional>
 Resource type. Possible values are:
export (for export presets), image, preset, script (for particles), shader, smartmask, smartmaterial, substance.
usages Array.<string> <optional>
 Tells in which context the resource is suited to be used. Possible values are the same as the ones available in the shelf GUI.
shelfName string Name of the shelf in which the resource is registered.
shelfPath string <optional>
 Path where the resource is located relative to the shelf root path.
tags Array.<string> <optional>
 Tags from a substance file.
category string <optional>
 Category from a substance file.
filePath string <optional>
 Resource file path. This attribute may not always be available.
