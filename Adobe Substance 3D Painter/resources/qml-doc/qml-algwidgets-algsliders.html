Detailed Description

A group of several sliders to edit vectors value with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgSliders {
  id: root
  dimensions: 4
  value: [0,0,0,0]
  minValue: [-1,-1,-1,-1]
  maxValue: [1,1,1,1]
}

Property Documentation

dimensions : int

The number of slider that the group must contain.

By default, dimensions is set to 1.


maxValue : var

The maximum values of each slider of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If maxValue.length < dimensions: the default value of the AlgSlider will be used.

By default, maxValue is empty.


minValue : var

The minimum values of each slider of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If minValue.length < dimensions: the default value of the AlgSlider will be used.

By default, minValue is empty.


precision : int

The precision of each slider of the group.

Every slider will have the same precision.

By default, precision is set to 3.


text : var

The labels of each slider of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If text.length < dimensions: the default value of the AlgSlider will be used.

By default, text is set to ["X", "Y", "Z", "W"].


value : var

The values of each slider of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If value.length < dimensions: the default value of the AlgSlider will be used.

By default, value is empty.
