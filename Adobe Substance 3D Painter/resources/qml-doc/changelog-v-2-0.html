Changelog v2.0

Compatibility

AlgWidgets 2.0 module is almost fully compatible with AlgWidgets 1.0.

Only one type has been removed, the other types are fully compatible. (see Removed types)

Some types have been added. (see New types)

AlgWidgets style

All graphic items from AlgWidgets module have been updated to match the Substance Painter 2018.1.x style.

New types

New types have been added to the module:

New typeBrief description
AlgToolBarButtonThis new type can be used to easily create a button in the same style as Substance Painter toolbar buttons. It greatly simplifies the usage of alg.ui.addWidgetToPluginToolBar or alg.ui.addToolBarWidget.
AlgToolTipAreaThis new type can be used to add a mouse area that handle tooltips in a custom graphic item.
AlgWidgets.StyleThis internal module has been entirely reworked and is no more compatible with its first version. We don't recommend to use this module if users want to keep compatibility with older versions.

Removed types

AlgDefaultSlider has been removed from AlgWidgets module.
