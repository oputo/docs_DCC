Detailed Description

A dialog with Substance 3D Painter style. This class is part of the AlgWidgets module.

This dialog allows users to press escape to be closed and enter to accept its purpose.

import AlgWidgets 2.0
import QtQuick 2.7

AlgDialog {
  id: root
  width: 100
  height: 100
  x: 50
  y: 50
  visible: true
  Rectangle {
    anchors.fill: parent
    color: "red"
  }
}

Property Documentation

contentItem : Component

Area of the dialog that can be fully used to draw content.

Usage example:

AlgDialog {
  Rectangle {
    parent: contentItem
    anchors.fill: parent
    color: "green"
  }
}

defaultButtonText : string

Text used to fill the default button.

By default, defaultButtonText is set to "Save".


Signal Documentation

accepted()

Signal emitted when the user has clicked on the default button.

Note: The corresponding handler is onAccepted.


Method Documentation

accept()

Force the dialog to accept its purpose and to be closed.
