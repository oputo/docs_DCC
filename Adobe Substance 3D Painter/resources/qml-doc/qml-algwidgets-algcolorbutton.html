Detailed Description

A widget used to set an RGB color. This widget is part of the AlgWidgets module.

When clicking on the AlgColorButton, a popup will appear with the Substance 3D Painter default color picker. It is possible to switch between linear and sRGB mode.

import AlgWidgets 2.0

AlgColorButton {
  id: root
  color: "magenta"
}

Property Documentation

arrayColor : var

Actual RGB color that the widget refers to as an array of 3 reals.

It is possible to use this property to set the color:

AlgColorButton {
  arrayColor: [0.5, 0.5, 0.5]
}

By default, arrayColor is set to [0, 0, 0].


color : color

Actual RGB color that the widget refers to.

By default, color is set to #000.


[read-only] isEdited : bool

The state of the widget: if the color picker dialog is opened, isEdited will be true.


linearMode : bool

Determines if the color must be in linear mode or in sRGB mode.

By default, linearMode is set to true.
