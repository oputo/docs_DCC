Detailed Description

A toggle button with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgToggleButton {
  id: root
  text: "AlgToggleButton"
  onToggledChanged: {
    alg.log.warn("toggled: " + toggled)
  }
}

Property Documentation

toggled : bool

This property holds the toggled state of the button.

By default, toggled is set to false.
