Detailed Description

A group of several spinboxes to edit vectors value with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgSpinBoxes {
  id: root
  dimensions: 4
  value: [0,0,0,0]
  minValue: [-1,-1,-1,-1]
  maxValue: [1,1,1,1]
}

Property Documentation

dimensions : int

The number of spinbox that the group must contain.

By default, dimensions is set to 1.


maxValue : var

The maximum values of each spinbox of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If maxValue.length < dimensions: the default value of the AlgSpinBox will be used.

By default, maxValue is empty.


minValue : var

The minimum values of each spinbox of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If minValue.length < dimensions: the default value of the AlgSpinBox will be used.

By default, minValue is empty.


precision : int

The precision of each spinbox of the group.

Every spinbox will have the same precision.

By default, precision is set to 3.


singleStep : real

The step size of each spinbox of the group.

Every spinbox will have the same step size.

By default, singleStep is fixed to 10-precision.


value : var

The values of each spinbox of the group in a dimensions sized array.

The size of the array may be different than the dimensions value. If value.length < dimensions: the default value of the AlgSpinBox will be used.

By default, value is empty.
