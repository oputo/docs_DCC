Detailed Description

A combo box with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgComboBox {
  id: root
  model: [{text:"Alg", value:4}, {text:"Combo", value:5}, {text:"Box", value:6}]
  textRole: "text"
  onActivated: {
    alg.log.warn("Value for '" + model[index].text + "' is '" + model[index].value + "'")
  }
}
