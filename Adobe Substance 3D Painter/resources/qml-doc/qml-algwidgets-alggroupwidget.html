Detailed Description

A group of UI component. This group contains a toggle button to open or close the content added by the user. This class is part of the AlgWidgets module.

import AlgWidgets 2.0
import QtQuick.Layouts 1.3

AlgGroupWidget {
  text: "group widget"
  AlgSlider {
    Layout.fillWidth: true
    text: "slider"
  }
  AlgButton {
    Layout.fillWidth: true
    text: "button"
  }
}

Property Documentation

activeScopeBorder : boolean

This property allows to display a line around the content of the group widget.

By default, activeScopeBorder is set to false.


[default] children : List<Item>

Children of the AlgGroupWidget. A ColumnLayout is used to display the user content:

AlgGroupWidget {
  width: 400
  text: "group widget"

  AlgSlider {
    Layout.fillWidth: true
    text: "slider"
  }
  AlgButton {
    Layout.fillWidth: true
    text: "button"
  }
}

This example creates a group widget with a slider and a button inside. You must import QtQuick.Layouts to call Layout.fillWidth in order to fill the content of the group.

By default, children are empty.


text : string

The title of the AlgGroupWidget.

By default, text is empty.


toggled : boolean

This property holds the toggled state of the group widget.

By default, toggled is set to false.


tooltip : string

Tooltip text associated with the groupwidget.
