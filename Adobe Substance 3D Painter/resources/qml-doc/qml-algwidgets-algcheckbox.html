Detailed Description

A check box with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgCheckBox {
  id: root
  text: "AlgCheckBox"
  onCheckedChanged: {
    alg.log.warn("checked: " + checked)
  }
}

Property Documentation

elide : enumeration

See Text::elide for more informations.

By default, elide is set to Text.ElideNone.


[read-only] position : var

Enumeration of the possible positions of the checkbox text.

position.LeftThe text will be displayed in the left of the checkbox.
position.RightThe text will be displayed in the right of the checkbox.

textPosition : var

This property holds the position of the checkbox text.

By default, textPosition is set to position.Right.


tooltip : string

Tooltip text associated with the checkbox.


wrapMode : enumeration

See Text::wrapMode for more informations.

By default, wrapMode is set to Text.WordWrap.
