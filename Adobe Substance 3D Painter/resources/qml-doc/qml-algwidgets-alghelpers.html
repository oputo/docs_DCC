[read-only] dynamicWidget : QtObject

A set of functions to help managing dynamic widgets.

Function prototypeBrief description
function addImportPath(importPath)Add a path to the user defined import path list.
function removeImportPath(importPath)Remove a path from the user defined import path list.
function createWidget(widgetType, parent)Create a dynamic component.
  • widgetType: string holding the type of the widget that will be created.
  • parent: parent of the dynamic widget that will be created.

In order to use custom widgets, it is necessary to specify the import path when calling this function. A signal - importPathRequired - is emitted each time the function is called. It is possible to use this signal to update the import path list:

Connections {
  target: AlgHelpers.dynamicWidget
  onImportPathRequired: {
    AlgHelpers.dynamicWidget.addImportPath(Qt.resolvedUrl("."))
  }
}

Since the import path list is saved under this singleton, the update may be performed only once.
