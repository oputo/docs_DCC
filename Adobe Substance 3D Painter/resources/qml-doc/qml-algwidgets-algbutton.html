Detailed Description

A push button with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgButton {
  id: root
  text: "AlgButton"
  onClicked: {
    alg.log.warn("clicked")
  }
}

Property Documentation

elide : enumeration

See Text::elide for more informations.

By default, elide is set to Text.ElideNone.


iconName : string

Path to the optional icon to display

By default, iconName is empty.


iconSize : size

Size of the optional icon to display

By default, iconSize is undefined and a default squared will be used.


textAlignment : enumeration

Allow to change the text alignment inside the button. See Text::horizontalAlignment for more informations.

By default, textAlignment is set to Text.AlignHCenter.


tooltip : string

Tooltip text associated with the button.


wrapMode : enumeration

See Text::wrapMode for more informations.

By default, wrapMode is set to Text.WordWrap.
