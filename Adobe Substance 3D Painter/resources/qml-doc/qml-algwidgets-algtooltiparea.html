Detailed Description

A tooltip area that allow to display a tooltip with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0
import QtQuick 2.7

Rectangle {
  id: root
  color: "red"
  width: 100
  height: 50
  AlgToolTipArea {
    anchors.fill: root
    tooltip: "This is a red rectangle"
  }
}

Property Documentation

delay : int

Delay before displaying the tooltip (in ms).

By default, delay is set to 500.


[read-only] hovered : bool

Hovered state of the tooltip area.


tooltip : string

Tooltip text to display.
