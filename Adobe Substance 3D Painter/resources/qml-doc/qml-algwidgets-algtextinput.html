Detailed Description

A text input with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0
import QtQuick.Layouts 1.3

RowLayout {
  id: root
  AlgTextInput {
    Layout.preferredWidth: 100
    onFocusChanged: {
      alg.log.warn("edited: " + focus)
    }
  }
}

Property Documentation

backgroundColor : color

The optional background color.


borderActivated : bool

Determines if the text edit must be surround by a border.

By default, borderActivated is set to true.


borderColor : color

The optional border color.


borderOpacity : real

The optional border opacity.

By default, borderOpacity is set to 1.


tooltip : string

Tooltip text associated with the textinput.
