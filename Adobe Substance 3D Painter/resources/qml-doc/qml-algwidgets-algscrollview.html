Detailed Description

A scroll view with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0
import QtQuick 2.7

AlgScrollView {
  id: root
  width: 125
  height: 200
  Column {
    Repeater {
      model: 30
      delegate: AlgLabel {
        text: "Item at index: " + index
      }
    }
  }
}

Property Documentation

[default] children : List<Item>

Children of the AlgScrollView.

By default, children are empty.


viewportColor : color

This property holds the color of the viewport.

By default, viewportColor is transparent.


[read-only] viewportHeight : real

This property holds the height of the viewport.


[read-only] viewportWidth : real

This property holds the width of the viewport.
