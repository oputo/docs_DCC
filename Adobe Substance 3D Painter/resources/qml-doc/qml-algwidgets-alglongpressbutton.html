Detailed Description

A long press button with Substance 3D Painter style. This class is part of the AlgWidgets module.

When pressing the button without releasing it, a clicked event is sent periodically.

import AlgWidgets 2.0

AlgLongPressButton {
  id: root
  text: "AlgLongPressButton"
  iconName: "icon_path" // You can use a relative or absolute path here
  onClicked: {
    alg.log.warn("clicked")
  }
}

Property Documentation

interval : int

The interval between each event are sent in milliseconds.

By default, interval is set to 100.
