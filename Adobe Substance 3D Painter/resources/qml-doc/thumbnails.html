Resource thumbnails

Overview

It is possible to display resource thumbnails in a custom Substance 3D Painter plugin like the AlgResourceWidget do when a valid url is set. These thumbnails, originally displayed in the Substance 3D Painter interface like the shelf, can be retrieved with a special url when using an Image QML type.

The url must begin by "image://resources/" in order to ask the software for the thumbnail. Then, adding the url of a resource will allow an Image component to receive the right icon to display.

For example:

Image {
  property string url: ""
  source: "image://resources/" + url
}

This example will display an icon if the url property is set to a valid resource url.

Drag and drop

It is also possible to perform a simple drag and drop of your resource from the shelf to the Image component. There is no need to know the resource url since the drop event contains this information.

Here is a simple example:

Image {
  property string url: ""
  source: "image://resources/" + url
  // You can require a specific size for the
  // thumbnail by using sourceSize property
  sourceSize.width: 512
  sourceSize.height: 512

  DropArea {
    anchors.fill: parent
    onDropped: {
      if (drop.hasUrls) {
        parent.url = drop.urls[0]
        drop.accept()
      }
    }
  }
}
