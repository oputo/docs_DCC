Detailed Description

A slider with Substance 3D Painter style. This class is part of the AlgWidgets module.

It is composed of a standard slider and a tool button allowing the user to request or close the edition mode of the slider.

import AlgWidgets 2.0

AlgSlider {
  id: root
  value: 0
  minValue: -1
  maxValue: 1
  text: "AlgSlider"
}

Property Documentation

[read-only] isEdited : bool

Determines if the slider is currently on edition mode or dragged.


maxValue : real

The maximum value of the slider.

By default, maxValue is set to 1.


minValue : real

The minimum value of the slider.

By default, minValue is set to 0.


precision : int

The precision of the slider. The value represents the number of decimals that the slider value can have.

By default, precision is set to 3.


[read-only] roundValue : real

The current value rounded at the precision of the slider.


stepSize : real

The step size of the slider spinbox.

By default, stepSize is fixed to 10-precision.


text : string

The label of the slider.

By default, text is empty.


value : real

The current value of the slider.

By default, value is set to 0.
