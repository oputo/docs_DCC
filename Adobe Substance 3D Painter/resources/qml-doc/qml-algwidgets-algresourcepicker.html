Detailed Description

Allows to display the Substance 3D Painter resource picker. This class is part of the AlgWidgets module.

Usages

The resource picker is able to filter resources by usages using the following flags:

  • AlgResourcePicker.EMITTER
  • AlgResourcePicker.ENVIRONMENT
  • AlgResourcePicker.LUT
  • AlgResourcePicker.FILTER
  • AlgResourcePicker.GENERATOR
  • AlgResourcePicker.MASK
  • AlgResourcePicker.MATERIAL
  • AlgResourcePicker.PRESET_BRUSH
  • AlgResourcePicker.PRESET_LAYERS
  • AlgResourcePicker.PRESET_MASK
  • AlgResourcePicker.PRESET_MATERIAL
  • AlgResourcePicker.PRESET_PARTICLE
  • AlgResourcePicker.PRESET_TOOL
  • AlgResourcePicker.PROCEDURAL
  • AlgResourcePicker.RECEIVER
  • AlgResourcePicker.SHADER
  • AlgResourcePicker.TEXTURE

Property Documentation

filters : int

An integer containing the resource picker flags that must filter the content of the popup.

By default, filters is fixed to AlgResourcePicker.MATERIAL

See also Usages.


refineQuery : string

This property holds the popup search bar query.

By default, refineQuery is empty.


Signal Documentation

resourceSelected(string name, string previewID)

Signal emitted when a new resource has been selected.

Note: The corresponding handler is onResourceSelected.


Method Documentation

bool requestResourceFromUrl(string url)

Requests a new resource from an url. If the url is not valid, the previous one is kept. Returns true if the url is valid.


void show(point position)

Requests the Substance 3D Painter resource picker display at the screen position.
