Detailed Description

A button used to show the Substance 3D Painter resource picker. This class is part of the AlgWidgets module.

The widget allows the users to drag and drop resources on its content and to open a mini-shelf to pick a resource.

import AlgWidgets 2.0

AlgResourceWidget {
  id: root
  filters: AlgResourcePicker.MATERIAL | AlgResourcePicker.GENERATOR
}

Property Documentation

defaultLabel : string

This property holds the default label used of no resource is selected.

By default, defaultLabel is "Select texture".


filters : int

An integer containing the resource picker flags that must filter the content of the popup.

By default, filters is fixed to AlgResourcePicker.MATERIAL.

See also Usages.


refineQuery : string

This property holds the popup search bar query.

By default, refineQuery is empty.


[read-only] url : string

This property holds the url of the current resource set in the widget. If no resource is set, the url will be an empty string.


Method Documentation

requestUrl(url)

Requests a new resource from an url. If the url is not valid, the previous one is kept. Returns true if the url is valid.

Clear the previous url if url is undefined or empty.
