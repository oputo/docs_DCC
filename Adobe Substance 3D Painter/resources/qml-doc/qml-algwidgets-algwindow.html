Detailed Description

A window with Substance 3D Painter style. This class is part of the AlgWidgets module.

This window allows users to press escape to be closed.

import AlgWidgets 2.0

AlgWindow {
  id: root
  title: "AlgWindow"
  visible: true
}

Property Documentation

screenCentered : bool

Determines if the window must be created at the center of the screen. If set to false, the behavior is undefined.

By default, screenCentered is set to true.


Method Documentation

Close the window.


open()

reload()

Reload the content of the window, must be overriden.
