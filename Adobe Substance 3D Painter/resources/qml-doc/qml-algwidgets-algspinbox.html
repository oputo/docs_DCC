Detailed Description

A spinbox with Substance 3D Painter style. This class is part of the AlgWidgets module.

import AlgWidgets 2.0

AlgSpinBox {
  id: root
  value: 0
  minValue: -1
  maxValue: 1
}

Property Documentation

maxValue : real

The maximum value of the spinbox.

By default, maxValue is set to 10.


minValue : real

The minimum value of the spinbox.

By default, minValue is set to 0.


precision : int

The precision of the spinbox. The value represents the number of decimals that the spinbox value can have.

By default, precision is set to 3.


singleStep : real

The step size of the spinbox.

By default, singleStep is fixed to 10-precision.


tooltip : string

Tooltip text associated with the spinbox.


value : real

The value of the spinbox.

By default, value is set to 0.
