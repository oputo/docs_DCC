Insertion
---------

Insertion of a layer or an effect can be done by using one of the following functions.
For more details, see :doc:`layereffect/index`.

- :func:`substance_painter.layerstack.insert_paint`
- :func:`substance_painter.layerstack.insert_fill`
- :func:`substance_painter.layerstack.insert_generator_effect`
- :func:`substance_painter.layerstack.insert_filter_effect`
- :func:`substance_painter.layerstack.insert_levels_effect`
- :func:`substance_painter.layerstack.insert_compare_mask_effect`
- :func:`substance_painter.layerstack.insert_color_selection_effect`
- :func:`substance_painter.layerstack.insert_group`
- :func:`substance_painter.layerstack.instantiate`
- :func:`substance_painter.layerstack.insert_anchor_point_effect`
- :func:`substance_painter.layerstack.insert_smart_mask`
- :func:`substance_painter.layerstack.insert_smart_material`
