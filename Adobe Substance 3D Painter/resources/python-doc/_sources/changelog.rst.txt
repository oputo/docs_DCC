Changelog
=========

0.3.1 (Substance 3D painter 10.0.1)
------------------------------------

- :mod:`substance_painter.layerstack`
    - Fix :func:`substance_painter.layerstack.set_selection_type` to allow properties selection of an instance parent layer.
    - Fix :func:`substance_painter.layerstack.FillLayerNode.set_source`,
      :func:`substance_painter.layerstack.FillLayerNode.set_material_source`,
      :func:`substance_painter.layerstack.FillEffectNode.set_source` and
      :func:`substance_painter.layerstack.FillEffectNode.set_material_source`
      to forbid creation of cycles with anchor points.
      It will now throw a ``ValueError``.

- :mod:`substance_painter.source`
    - Fix :func:`substance_painter.source.SourceSubstance.set_source` to forbid creation of cycles with anchor points.
      It will now throw a ``ValueError``.
    - Add :func:`substance_painter.source.SourceBitmap.list_available_color_spaces` to query available colorspaces for a bitmap.

0.3.0 (Substance 3D painter 10.0.0)
------------------------------------

This version introduces new modules to be able to edit your project layerstack.
With this new modules you will be able to navigate, access, edit, remove and add every
type of layers and effects available in the UI of Substance Painter.

- :mod:`substance_painter.layerstack`
    - New module to manipulate the layer stack.

- :mod:`substance_painter.source`
    - New module to manipulate layer's sources.

- :mod:`substance_painter.colormanagement`
    - New module with colormanagement facilities for layerstack edition.


- :mod:`substance_painter.resource`
    - Add ability to get substance presets details using :func:`substance_painter.resource.Resource.internal_properties` function.

- :mod:`substance_painter.export`
    - Add :class:`substance_painter.export.PredefinedExportPreset` and :func:`substance_painter.export.list_predefined_export_presets`
      to retrieve predefined export presets and list their output maps.
    - Add :class:`substance_painter.export.ResourceExportPreset` and :func:`substance_painter.export.list_resource_export_presets`
      to retrieve resource export presets (editable presets from shelves) and list their output maps.

0.2.12 (Substance 3D painter 9.1.1)
------------------------------------

- :mod:`substance_painter.project`
    - Add :func:`substance_painter.project.get_uuid` to query the project UUID.

0.2.11 (Substance 3D painter 9.1.0)
-----------------------------------

- :mod:`substance_painter.application`
    - Add ability to disable engine computations
    - Add :func:`substance_painter.application.close` to close Substance 3D Painter.

- :mod:`substance_painter.display`
    - Add :class:`substance_painter.display.Camera` to manipulate camera settings.

- :mod:`substance_painter.event`
    - Add a new event :class:`substance_painter.event.EngineComputationsStatusChanged` to be notified when engine computations status changes.
    - Add a new event :class:`substance_painter.event.CameraPropertiesChanged` to be notified when the properties of a camera change.

- :mod:`substance_painter.export`
    - Add ability to export the mesh of the current project.
    - Add ability to get the default export path used for exporting textures.

- :mod:`substance_painter.project`
    - Add :class:`substance_painter.project.BoundingBox` and :func:`substance_painter.project.get_scene_bounding_box` to query dimensions of the scene.

0.2.10 (Substance 3D painter 9.0.0)
-----------------------------------

- :mod:`substance_painter.export`
    - Fix textures exported in EXR with 32f bit depth were incorrectly saved in 16f.

- :mod:`substance_painter.project`
    - Add :class:`substance_painter.project.UsdSettings` in :class:`substance_painter.project.Settings` to support USD parameters for project creation.
    - Add :class:`substance_painter.project.UsdSettings` in :class:`substance_painter.project.MeshReloadingSettings` to support USD parameters for mesh reloading.

0.2.9 (Substance 3D Painter 8.3.1)
----------------------------------

- :mod:`substance_painter.async_utils`
    - New module to expose primitives used in async computations.
- :mod:`substance_painter.event`
    - Add a new event :class:`substance_painter.event.BakingProcessProgress` to be notified about baking process progress.
    - Add a new event :class:`substance_painter.event.BakingProcessAboutToStart` to be notified of the start of a baking process.
    - Allow to configure throttling period for :class:`substance_painter.event.TextureStateEvent`.
- :mod:`substance_painter.baking`
    - Add :func:`substance_painter.baking.bake_selected_textures_async` to bake all selected texture set.
    - Fix :func:`substance_painter.baking.bake_async` to take UV Tiles selection into account.
    - Now async functions return a :class:`substance_painter.async_utils.StopSource` to allow for cancellation.

0.2.8 (Substance 3D Painter 8.3.0)
----------------------------------

- :mod:`substance_painter.application`
    - New module to expose functionnalities on the application level.
    - Add :func:`substance_painter.application.version_info` to query Substance 3D Painter version as a tuple.
    - Add :func:`substance_painter.application.version` to query Substance 3D Painter version as a string.
- :mod:`substance_painter.event`
    - Add a new event :class:`substance_painter.event.BakingProcessEnded` to be notified of the end of the baking process.
- :mod:`substance_painter.project`

- :mod:`substance_painter.textureset`
    - Add :func:`substance_painter.textureset.TextureSet.uv_tile` to get a specific UV Tile.
- :mod:`substance_painter.ui`
    - Add Baking to :attr:`substance_painter.ui.UIMode`.
    - Add :func:`substance_painter.ui.get_current_mode` to query the current UI mode.
    - Add :func:`substance_painter.ui.switch_to_mode` to switch to some UI mode.

This version introduces the modules:

- :mod:`substance_painter.baking`
- :mod:`substance_painter.properties`

0.2.7 (Substance 3D Painter 8.2.0)

    - Add a new event :class:`substance_painter.event.TextureStateEvent` to be notified of document textures edition.

0.2.5 (Substance 3D Painter 8.1.0)
----------------------------------

This version upgrades to Python 3.9.9.

- :mod:`substance_painter.project`
    - :func:`substance_painter.project.create` supports PAINTER_ACE_CONFIG
      environment variable, to setup the project color management mode.
    - :func:`substance_painter.project.create` now raise an error if OCIO
      or PAINTER_ACE_CONFIG environment variable is set to an invalid
      configuration.

0.2.4 (Substance 3D Painter 7.4.3)
----------------------------------

This version exposes the tone mapping operator to the Python API, and adds
a hook to execute code when Substance 3D Painter is not busy.

- :mod:`substance_painter.display`
    - Add :func:`substance_painter.display.get_tone_mapping` to query the
      project tone mapping.
    - Add :func:`substance_painter.display.set_tone_mapping` to set the
      project tone mapping.
- :mod:`substance_painter.project`
    - Add :func:`substance_painter.project.execute_when_not_busy` to execute
      code when Substance 3D Painter is no longer busy.


0.2.3 (Substance 3D Painter 7.4.0)
----------------------------------

This version makes it possible to work with JavaScript from the Python API.
It also allows creating projects with OCIO color management capabilities.

- :mod:`substance_painter.js`
    - Add :func:`substance_painter.js.evaluate` to evaluate some JavaScript code.
- :mod:`substance_painter.project`
    - :func:`substance_painter.project.create` supports OCIO environment variable,
      to setup the project color management mode.


0.2.2 (Substance 3D Painter 7.3.0)
----------------------------------

This version adds functions to help with scripted project loading, project
mesh reloading and UV Tiles manipulation. It also improves error handling
in several modules.

- :mod:`substance_painter.display`
    - :func:`substance_painter.display.set_color_lut_resource` and
      :func:`substance_painter.display.set_environment_resource` throw a
      TypeError instead of a ValueError when an argument has a type different
      from expected.
- :mod:`substance_painter.event`
    - Add :func:`substance_painter.event.Dispatcher.connect_strong` to connect
      strong reference callbacks to the event dispatcher.
- :mod:`substance_painter.project`
    - Add :func:`substance_painter.project.is_busy` to query whether Substance
      3D Painter is busy.
    - Project saving functions are disabled when Substance 3D Painter is
      busy.
    - Add :func:`substance_painter.project.last_imported_mesh_path` to query the
      path to the last imported mesh.
    - Add :func:`substance_painter.project.reload_mesh` to load a new mesh in the
      current opened project.
    - :func:`substance_painter.project.create` throws a TypeError instead
      of a ValueError when its settings argument has a type different from
      expected.
- :mod:`substance_painter.resource`
    - Add :func:`substance_painter.resource.show_resources_in_ui` to highlight a
      list of resources in the aplication UI.
    - :func:`substance_painter.resource.update_layer_stack_resource` throws
      a TypeError when an argument has a type different from expected.
- :mod:`substance_painter.textureset`
    - Add :class:`substance_painter.textureset.UVTile` to interact with UV Tiles.
      Add ability to manipulate UV Tiles resolutions.
    - :func:`substance_painter.textureset.TextureSet.from_name` throws a
      TypeError instead of a ValueError when its argument has a type different
      from expected.
