# Uma solução de design 3D conectada
O Adobe Substance 3D Designer permite criar materiais com controle total. É uma parte de um ecossistema de aplicativos e conteúdo que permite que você crie tudo em 3D.

# Criação processual
Crie materiais e padrões processuais, filtros de imagem e luzes de ambiente. Conecte nós para gerar variações infinitas em cada ativo e traga-os para outras ferramentas 3D.

![visual](feature_2.png)

# Um ecossistema de materiais completo
Suas criações do Designer são compatíveis com praticamente qualquer fluxo de trabalho. A grande maioria dos aplicativos 3D suporta materiais do Substance, para que você possa personalizá-los onde quer que precise.

![visual](feature_3.png)

# O padrão para materiais 3D
O Substance 3D Designer é um editor baseado em nós. Ele permite que você crie materiais de procedimento de uma forma não linear e não destrutiva, e exponha qualquer parâmetro para que os usuários finais possam controlar.
